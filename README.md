ibis-rb
=======

Ruby port of [Ibis](https://github.com/takuto-h/ibis-js) (WIP)

Requirements
------------

Ruby >= 2.1 (mainly for .to_h)

How to run
----------

    $ bundle install
    $ bundle exec rake run   # exec main.rb
    $ bundle exec rake test  # run unit tests
